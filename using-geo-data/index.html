Ogma geo mode is:

  1. A simple API
  2. Based on leaflet
  3. Export feature
  4. Group feature

API

            
              ogma.geo.enable()

API

            
              ogma.geo.enable({
                tiles:{
                  url:"//{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}{r}.png",
                }
              })
  1. Use any map provider (WMS, cartocdn, local, public...)

API

            
                ogma.geo.disable()

Coordinates

  1. WebMercator projection (EPSG3857)
  2. Stored by node
  1. Custom, non geographic, local projections possible
  2. Nodes without coods ignored

Coordinates

            
              ogma.geo.enable({
                latitudePath: "lat",
                longitudePath: "long"
              })
  1. Path is flexible, dot notation
  2. Nodes without coods ignored

Coordinates

            
              ogma.getNodes().get(0).setData('lat', 48.858838);
              ogma.getNodes().get(0).setData('long', 2.343436);
