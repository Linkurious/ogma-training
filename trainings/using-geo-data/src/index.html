Les cartes avec

Ogma

Ogma geo mode c'est:

  1. Une API simple
  2. Basée sur leaflet
  3. Possibilité d'export
  4. Possibilité de grouper

API

            
              ogma.geo.enable()

API

            
              ogma.geo.enable({
                tiles:{
                  url:"//{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}{r}.png",
                }
              })
  1. Utiliser n'importe quel map Providers (WMS, cartocdn, local, public...)

API

            
                ogma.geo.disable()

Coordonées

  1. Projection WebMercator (EPSG3857)
  2. Stockées par node
  1. Custom, non geographic, local projections possible
  2. Nodes sans coords pas afffiches

Coordonées

            
              ogma.geo.enable({
                latitudePath: "lat",
                longitudePath: "long"
              })
  1. Path est flexible, dot notation
  2. Nodes sans coords pas afffiches

Coordonées

            
              ogma.getNodes().get(0).setData('lat', 48.858838);
              ogma.getNodes().get(0).setData('long', 2.343436);
