Les cartes avec

Ogma

Ogma geo mode c'est:

  1. Une API simple
  2. Basée sur leaflet
  3. Possibilité d'export
  4. Possibilité de grouper

API

            
              ogma.geo.enable()

API

            
              ogma.geo.enable({
                tiles:{
                  url:"//{s}.basemaps.cartocdn.com/light_all/{z}/{x}/{y}{r}.png",
                }
              })

API

            
                ogma.geo.disable()

Demo

Coordonées

  1. Projection Mercator (EPSG3857)
  2. Stockées par node

Coordonées

            
              ogma.geo.enable({
                latitudePath: "lat",
                longitudePath: "long"
              })

Coordonées

            
              ogma.getNodes().get(0).setData('lat', 48.858838);
              ogma.getNodes().get(0).setData('long', 2.343436);
